# Portal de Autenticación

Este es un proyecto personal que recrea un sistema de **inicio de sesión y gestión de usuarios** inspirado en plataformas e-learning. El objetivo es demostrar la implementación de flujos de autenticación modernos y una interfaz de usuario funcional.

## 🚀 Funcionalidades Implementadas

Basado en el diseño del portal, el proyecto incluye:

*   **Acceso Empresarial (SSO):** Interfaz para inicio de sesión mediante correo corporativo.
*   **Autenticación Tradicional:** Formulario de acceso mediante usuario y contraseña.
*   **Recuperación de Acceso:** Flujo diseñado para usuarios que han olvidado sus credenciales.
*   **Sistema de Verificación:** Herramienta para la resolución de problemas de acceso y validación de cuentas.

## 🛠️ Tecnologías Utilizadas
Para el desarrollo de este proyecto se han empleado las siguientes tecnologías (información proporcionada por el autor, externa a las fuentes):
###  Frontend:
*  HTML: Estructura principal de las vistas y formularios.
*  CSS: Estilos y maquetación visual.
###  Backend:
*  Java: Lógica de negocio y procesamiento de datos.
*  Spring Boot: Framework base para la configuración y seguridad del proyecto.
###  Gestión de Dependencias:
*  Maven: El proyecto utiliza un archivo pom.xml y los ejecutables mvnw para la gestión de construcción y dependencias.

## 📝 Descripción del Proyecto
Este repositorio contiene el código fuente desarrollado para simular un portal de acceso seguro. Se ha prestado especial atención a la disposición de elementos legales y de soporte, tales como:
*   Enlaces a Términos y Condiciones.
*   Secciones de Política de Privacidad y Cookies.
*   Documentación de Política de Seguridad.
